Adventure in the Iron Laboratory

Back to Main Page

Release date: 11.01.2021

On this map, you have to go through different tests (mostly parkour). Your walkthrough will be accompanied by a computer. Its name is "Petya 2.1".

Interesting fact: When building the map, I was inspired by the game Portal 2.

Map features:

+ Achievements

+ Soundtrack from different games (can be disabled)

+ Datapack

+ Mods

How install map:

1. Install forge.

2. Download archive.

3. From archive unzip folder "Adventure in the Iron Laboratory" in ".minecraft\saves".

4. In the archive, unpack everything from the "mods" folder to the ".minecraft\mods" folder.
