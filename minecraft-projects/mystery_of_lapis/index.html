Mystery of Lapis

Back to Main Page

On this map, you have to play as Jerry. He was an ordinary person, lived alone. After a summer vacation, Jerry returns to work again. However, when he entered the building, he did not see a soul. His first task is to find everyone, but he still does not know what awaits him!

Help:

If you have bacchanalia on the signs, use the command "/trigger reload_signs".

If you are stuck in the computer, write the command "/trigger exit_computers".

How install map:

1. Download arcive.

2. Unzip the folder "mods" into a .minecraft folder.

3. Unzip folder "Mystery of Lapis" into .minecraft/saves folder.
